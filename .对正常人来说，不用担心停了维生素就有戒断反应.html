对正常人来说，不用担心停了维生素就有戒断反应

图片：Colin Dunn / CC BY

长时间服用维生素等营养补充剂后突然停止服用，会有「戒断反应」吗？

KellyWeaver，For Public Health

（未经我的同意禁止转载（日报除外））

先上结论：

目前来看，对于营养状态正常的人来说，没有证据表明服用维生素补充剂后突然停止服用会产生“戒断综合征”。题主所说的情况更有可能是心理原因造成的，尤其是免疫力这种受情绪影响这么大的东西=，=（还有就是，题主你对免疫力的定义是什么？难道是一天没精神？鉴于你说的“放假后”这么感觉，我觉得更可能的原因是你没有睡好……）

目前没有任何证据表明哪种维生素在你平时摄入足够的情况下再去补充再突然停药会给你找事儿。即使是维生素 C，也不会出现所谓的“反跳性坏血病”。

本来就严重缺乏维生素 C 的人，如果不改正膳食不合理之处，就算不来上吃补剂再突然停药这么一出，也会发展成坏血病。

看答案里大家都在说维生素 C 和反跳性坏血病，这是从哪里听说的？中文文章里面倒是搜到两篇这么讲，无一例外没有引用资料。劝人不要乱吃维 C 不是这么劝的啊喂。

根据经验，一般这类中文科研界没人讨论的结论基本上都是从国外某一篇耸人听闻的文章翻译来的。我去查了一下英文的文章，发现你们所说的这个“反跳性坏血病”的概念应该是来自于这里[1]。但是，这篇文章里面提到的实际上是母亲摄入维生素 C 和婴儿的坏血病的关系，而且这篇文章里面的母亲也就每天吃一片维生素 C，而近些年基本上全美 90% 的母亲都报告说自己在孕期服用维生素 C[2]，也没见美国人生下来集体坏血病啊。

针对这个事情，有学者专门做了一个实验，给一群平时饮食中维生素 C 摄入充足的人吃维生素 C 补剂，然后突然停药，然后——啥都没有发生。大家血液中的维 C 含量仅仅比他们吃维生素 C 补充剂之前的水平低了那么一丁点而已，几乎可以忽略[3]。

针对本身严重缺乏维生素 C 的人，如果缺乏了之后不改正自己错误的饮食习惯，反倒是寻求维生素 C 补充剂，那么突然停药倒的确是会引起血液白细胞维生素 C 含量显著下降[4]。不过这个现象倒还不止出现在维生素补剂上，就算是吃一般饮食的正常人，如果你突然把他饮食中维 C 的来源全砍了，过不了三周时间这人也会出现这么一个“反跳性下降”的效应[5]。

而且，很有可能其实你不需要补充维 C。中国人民作为一个有着健康饮食天赋的群体，本来蔬菜水果就吃得多，目前维生素 C 摄入的均值还是算不错的。维生素 C 的推荐摄入量（RNI）是 100mg/d，平均需要量（EAR）是 85mg/d [6]。目前我国成人的日摄入量均值，城市 82.2mg/d，农村 90mg/d [7]，虽然还有一点儿距离，需要加强，但是从与 EAR 的距离来看，应该还是不会有很多人出现缺乏。

目前临床那边好像也是看见文献说，平时工作中很少见维生素 C 缺乏来看病的。基本上，要有都是很轻度的缺乏，据说如果有重症缺乏搞出坏血病的，还要叫上轮转的实习医生赶紧过来观摩。具体证据不太清楚，希望临床的大牛补充~

所以，总结起来就是，维生素 C 这玩意儿，如果你要是真缺（而不是你觉得你缺），就要多加利用自己的天赋加成，靠自己国家的健康饮食结构和多样的膳食来源好好吃。不要总想着吃补剂，否则到头来还给自己找一堆事儿。
